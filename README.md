# Tony Dashboard

Mission Control for Dave & Tony.

**Live URL:** https://brain.dbcrypt0.com (to be configured)

## What This Is

A visual dashboard for managing OpenClaw workspace, tracking goals, monitoring systems, and maintaining accountability.

## Features (Planned)

- **System Status:** Current LLM model, uptime, installed skills, cron jobs
- **Memory Viewer:** Quick access to MEMORY.md, SOUL.md, USER.md, daily logs
- **Goal Tracker:** Revenue goals, weekly targets, accountability metrics
- **Content Pipeline:** Integration with Crevara for video/content tracking
- **Analytics:** Token usage, model costs, activity patterns
- **Quick Actions:** Model switching, manual compaction, session management

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Firebase (or alternative backend)
- OpenClaw API integration

## Development

```bash
npm install
npm run dev
```
